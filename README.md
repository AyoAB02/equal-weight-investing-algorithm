# Equal-Weight Investing Algorithm

An equal-weight portfolio optimisation algorithm that constructs diversified equity portfolios using real-time market data, with automated position sizing based on portfolio value.

## Overview

This algorithm takes a universe of securities and constructs an equal-weight portfolio, calculating the number of shares to purchase for each position based on a specified portfolio size. Unlike market-cap-weighted indices (where a few large companies dominate), equal weighting provides more balanced exposure across all holdings.

## Tools & Libraries

- **Python** (pandas, NumPy, yfinance)
- Real-time market data via Yahoo Finance API
- Automated position sizing and rebalancing calculations

## Key Features

- Fetches live market data for any user-defined stock universe
- Calculates equal-weight allocations based on total portfolio value
- Outputs recommended share quantities per position
- Handles portfolio rebalancing calculations
